| Licencia | Descripción | Ejemplos |
| :--- | :--- | :--- |
| **GNU GPL** | Licencia de software libre con **copyleft**. Obliga a que cualquier obra derivada mantenga la misma licencia y sea de código abierto. | Linux, WordPress, GIMP |
| **MIT** | Licencia **permisiva** y minimalista. Permite uso comercial y privado con la única condición de incluir el aviso de copyright original. | React, Node.js, VS Code |
| **Apache 2.0** | Permisiva, similar a MIT pero añade una **concesión expresa de derechos de patentes** y requiere documentar cambios importantes. | Android, Kubernetes, TensorFlow |
| **BSD** | Licencia **académica permisiva**. Permite la redistribución y el uso en software propietario sin obligar a compartir el código fuente. | FreeBSD, Go, Chromium |
| **Mozilla (MPL)** | Licencia **híbrida**. El código original debe ser abierto, pero permite combinarlo con archivos bajo licencias propietarias. | Firefox, LibreOffice |
| **Propietaria** | El autor retiene todos los derechos. El usuario solo recibe una licencia de uso limitada, **sin acceso al código fuente**. | Windows, Adobe Photoshop |
